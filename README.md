# 个人动态网站 & 留言板

这是一个我自主构建的个人网站，集成了前端展示与后端数据交互功能。

## 🚀 功能特性
- **个人主页**：展示个人介绍。
- **云端留言板**：访客留言可永久存储在云端数据库，并实时展示。

## 🛠️ 技术栈
- **前端**：HTML, CSS (Tailwind CSS), JavaScript
- **后端/数据库**：Supabase (PostgreSQL)
- **部署**：GitHub Pages
